Nabel plot

















This page is an example of an OpenCPU app. It it part of the opencpu.demo package and uses the R-function nabel.

The nabel function create a lattice plot from live data provided by the Swiss National Air Pollution Monitoring Network.. See also this blog post about this app. For the source code, help file, or other information, click on one of the links below.
